# ESP32摄像头连接指南

## 问题描述

当点击ESP32摄像头按钮时，无法切换到ESP32摄像头，仍然显示电脑本地摄像头。

## 问题原因

ESP32摄像头无法连接，主要原因如下：

1. **网络连接问题**：电脑没有连接到ESP32创建的WiFi网络
2. **摄像头地址错误**：ESP32摄像头地址可能不正确
3. **设备未启动**：ESP32设备可能未开机或未运行摄像头程序

## 解决方案

### 第一步：连接ESP32的WiFi网络

1. **确保ESP32设备已开机**：
   - 检查ESP32设备电源指示灯是否亮起
   - 确认ESP32已正确连接摄像头模块

2. **搜索ESP32的WiFi热点**：
   - 在电脑上打开WiFi设置
   - 搜索以"ESP32"或"ESP32-CAM"开头的WiFi网络
   - 常见的WiFi名称可能包括：
     - `ESP32-CAM`
     - `ESP32_AP`
     - `AI-Thinker`
     - 或其他自定义名称

3. **连接到ESP32的WiFi网络**：
   - 选择ESP32的WiFi网络并连接
   - 如果需要密码，常见的默认密码包括：
     - `12345678`
     - `password`
     - 或查看ESP32程序中的设置

4. **验证连接成功**：
   - 连接成功后，电脑应该显示"已连接"状态
   - 可以尝试在浏览器中访问 `http://192.168.4.127/` 测试连接

### 第二步：检查ESP32摄像头地址

1. **默认摄像头地址**：`http://192.168.4.127/`
2. **可能的其他地址格式**：
   - `http://192.168.4.127/capture`
   - `http://192.168.4.127/cam.jpg`
   - `http://192.168.4.127/cam.mjpeg`
   - `http://192.168.4.127/stream`

3. **修改摄像头地址**：
   - 在网页界面的"ESP32摄像头地址"输入框中修改地址
   - 如果默认地址无效，尝试其他格式

### 第三步：测试连接

1. **运行测试脚本**：
   ```bash
   python test_esp32_camera.py
   ```

2. **在浏览器中测试**：
   - 打开新的浏览器标签页
   - 输入ESP32摄像头地址（如 `http://192.168.4.127/`）
   - 如果显示图像，说明连接成功

### 第四步：在应用中切换摄像头

1. **确保已连接到ESP32的WiFi网络**
2. **打开应用**：`http://127.0.0.1:5000`
3. **点击"目标检测"标签**
4. **点击"ESP32摄像头"按钮**
5. **观察连接状态**：
   - 如果显示"正在连接ESP32摄像头..."，说明正在尝试连接
   - 如果显示"ESP32摄像头连接失败"，请检查上述步骤
   - 如果成功连接，canvas区域将显示ESP32摄像头的实时图像

## 常见问题排查

### 问题1：无法找到ESP32的WiFi网络
- **原因**：ESP32设备未开机或WiFi功能未启用
- **解决方案**：检查ESP32设备状态，重新启动设备

### 问题2：连接WiFi后无法访问摄像头
- **原因**：摄像头地址错误或ESP32程序未运行
- **解决方案**：
  1. 检查ESP32程序是否正确上传和运行
  2. 尝试不同的摄像头地址格式
  3. 重启ESP32设备

### 问题3：连接成功但图像不显示
- **原因**：浏览器安全策略或跨域问题
- **解决方案**：
  1. 确保使用现代浏览器（Chrome、Firefox等）
  2. 检查浏览器控制台是否有错误信息
  3. 尝试清除浏览器缓存

### 问题4：图像显示但检测功能不工作
- **原因**：网络延迟或图像质量问题
- **解决方案**：
  1. 确保网络连接稳定
  2. 检查ESP32摄像头图像质量
  3. 调整检测间隔时间

## 技术说明

### 网络架构
```
ESP32设备 → WiFi热点 → 电脑浏览器 → Flask应用
    ↓
摄像头图像 → HTTP传输 → Canvas显示 → YOLOv8检测
```

### 连接流程
1. ESP32创建WiFi热点
2. 电脑连接到ESP32的WiFi网络
3. 浏览器通过HTTP请求获取摄像头图像
4. 图像显示在Canvas元素中
5. 应用定期捕获Canvas帧进行目标检测
