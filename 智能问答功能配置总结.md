## 使用说明

### 网页端使用

1. 访问 http://127.0.0.1:5000
2. 点击"智能问答"菜单项
3. 在文本框中输入问题
4. 点击"提交"按钮
5. 系统会显示回答结果（目前由于网络问题会显示错误信息）

### API调用

```python
import requests

data = {
    "question": "你的问题",
    "image_url": None  # 可选，图片URL
}

response = requests.post("http://127.0.0.1:5000/api/ask", json=data)
result = response.json()
```
